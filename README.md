# vingle-internship

vingle

## Before execution

should run the mysql instance and create database using below command

```
create database vingle;
```

## Install It
```
npm install
```

## Run It
#### Run in *development* mode:

```
npm run dev
```

#### Run in *production* mode:

```
npm run compile
npm start
```

#### Run tests:

```
npm test
```
